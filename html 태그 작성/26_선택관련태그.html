select / option 태그

        - select : 클릭하면 밑으로 메뉴가 내려오는 입력 태그
        - option : select 클릭시 보여질 메뉴를 작성하는 태그

기타 입력 관련 태그

        - type="color" : 색을 선택할 수 잇는 입력상자 출력
        - type="file" : 파일을 선택할 수 잇는 탐색기 진행
        - type="hidden" :  화면에 보이지 않음
                            (화면에서 숨기고 싶은 값이 있을 경우 사용)
